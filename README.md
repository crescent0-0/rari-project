# `/dashboard` : 실시간 이커머스 운영 대시보드

1. 개요
- 목표 : 주문·매출·전환율·장바구니 이탈·오류율을 “초 단위”로 보여주고, 임계치 초과 시 알림 및 원인 탐색(드릴다운) 지원.
- 핵심 가치:
    - 초저지연 실시간 반영 (Rari + SSE/WebSocket)
    - 서버 컴포넌트 기반의 빠른 초기 로드 & 부분 스트리밍
    - 쉬운 운영(무중단 배포, 대시보드 구성 저장)

2. 기능목록
- KPI 실시간 카드 : 매출(₩/분), 주문수(건/분), 전환율(%), 평균 주문금액(AOV), 실패 결제율
- 실시간 트래픽 차트 동시 접속자, 초당 페이지뷰(PV/s), 채널별(Direct/SEO/AD) 추세

- Top 상품/카테고리: 매출/수량 기준 상위 N개, 클릭 시 상세 드릴다운

- 이상 감지 & 알림: 임계치 설정(예: 실패 결제율 > 3% 3분 지속) → UI 배지 + Slack/Webhook 알림

- 오류 피드 :최근 100건의 결제/주문 오류(코드·메시지·엔드포인트·상태)

- 시간 범위 & 라이브 토글

= 시간(초 단위) 모드 ↔ 과거 구간 재생(리플레이)

-  어 기반 접근 제어
: Admin / Operator / Viewer
